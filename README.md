## What is this
ngvm2 is a tool to manage multiple versions of Angular.
It is inspired by [nvm]() installing multiple versions of Node.js.
## Usage
### Install
```bash
npm install -g ngvm2
```
### Usage
```bash
ngvm2 install 1.6.0
ngvm2 use 1.6.0
```
